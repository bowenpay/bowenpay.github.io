每个月的工资：6.6万

房租（含水电天然气）：1万

网费：500元

每个商户的成本 5000元，现在一个月3家左右，每月支出是： 1.5万

每个用户的成本 20元，每个商家能带来300个用户，每月用户成本是：20 * 3 * 300 = 1.8万元

用户数每月是：3 * 300 = 900个

每月固定支出：66000 + 10000 + 500 + 15000 + 18000 = 10.95万

所以按照现在的速度，当前的用户成本是：10.95万 / 900 = 122 元

那么如果我们手上有200万，能够支持发展多久： 200 / 10.95 = 18个月

发展多少商户：18 * 3 = 54家

获取多少用户：900 * 18 = 1.62万
